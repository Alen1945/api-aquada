# API AQUADA

API FOR APP AQUADA - Water Delivery And Order

#### Installation

```shell
$ git clone ""
$ cd 
$ rm -rf .git
$ npm i
$ npm i nodemon && npm i sequelize-cli
$ sequelize db:migrate && sequelize db:seed:all
$ npm start
```

Secara default akan terbuka port 6000 atau http://localhost:6000
